# 方法论决策

> 本文档记录实验设计中每个决策的理由。所有"事先定死"的约定,
> 以本文件的 git 时间戳为准。

## 判定门槛(在跑增强实验之前定死)

Baseline(事件划分,5 seeds):macro_f1 = 0.705 ± 0.010

### 可分辨门槛
单次运行 σ ≈ 0.010,5 次均值的 σ ≈ 0.0045,两组均值之差的 σ ≈ 0.0063。
故本实验能分辨的最小差异约为 **0.013**。

低于此值时,正确的表述是"在本实验精度下未观察到显著差异",
而不是"没有差异"—— 前者是关于仪器的断言,后者是关于世界的断言,
本实验只赚到前者。

### 判定标准
**可分辨门槛 = 值得在乎门槛 = 0.013。**

理由:本实验问的是"对立结构起不起作用",不是"这个方法值不值得部署"。
前者只需要效应可靠可分辨;成本论证(生成 6406 条论证需 $5 + 4 小时)
属于后者,不在本实验射程内。

代价:若结果为 +0.015,只能说"检测到一个小而真实的效应",
**不能说"这是个好方法"** —— 后半句需要的证据本实验没有。

### 主指标
**主指标:macro_f1。门槛 0.013。**

rumour_recall 是探索性观察,可以报告,但不能用来宣称方法有效。
理由:指标越多,总有一个在动(garden of forking paths)。
若 macro_f1 无变化而 recall 上升,那是一个值得追的线索,不是一个结论。

### 分析方法
Baseline 与增强版共用同一组 seed(配对设计),因此比较的是
5 个逐 seed 的差值 d_i = aug_i - base_i 的分布,而非两组均值。
配对可消去"某个 seed 天生跑得好"这部分噪声。
